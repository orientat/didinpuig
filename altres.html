👮 Policia Nacional

Requisits: nacionalitat espanyola, +18 anys, títol de Batxiller o equivalent, carnet B.
Proves: físiques (resistència, agilitat), examen teòric, psicotècnics, entrevista i reconeixement mèdic.
🎯 ESO → Batxillerat → oposició → Acadèmia d’Àvila
💼 Salides: seguretat ciutadana, investigació criminal, unitats especials.
✖

🌍 Idiomes (EOI)

Accés a partir de 16 anys. Prova de nivell inicial.
Nivells oficials: A2 · B1 · B2 · C1 · C2
Certificats oficials vàlids per universitat, oposicions i treball.
🎯 ESO / Batx → EOI → certificat oficial → universitat / treball
💼 Salides: estudis internacionals, oposicions, treball amb idiomes.
✖

🎵 Música i Dansa

Estudis en conservatoris professionals amb prova específica.
🎯 ESO → prova → conservatori → estudis superiors
💼 Salides: músic, docent, arts escèniques.
✖

🖌️ Arts Plàstiques i Disseny

Dibuix, fotografia, disseny gràfic, il·lustració.
🎯 ESO → prova → escola d’art → grau superior
💼 Salides: dissenyador, artista, creatiu digital.
✖

⚽ Ensenyaments Esportius

Formació de tècnic esportiu amb prova física.
🎯 ESO → prova → tècnic esportiu
💼 Salides: entrenador, monitor esportiu.
