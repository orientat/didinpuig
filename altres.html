👮 Policia Nacional

Requisits: nacionalitat espanyola, +18 anys, Batxiller o equivalent, carnet B.
Proves: físiques (resistència, agilitat), teòriques, psicotècnics, entrevista i mèdiques.
🎯 ESO → Batxillerat → oposició → Acadèmia d’Àvila
💼 Seguretat ciutadana, investigació criminal, unitats especials.
Web oficial
✖

🛡️ Guàrdia Civil

ESO o superior, proves físiques i teòriques.
Especialitats: trànsit, muntanya, medi ambient (SEPRONA).
🎯 ESO → FP/Batx → oposició
💼 Seguretat, investigació, protecció mediambiental.
Web oficial
✖

🏙️ Policia Local

Depén de cada ajuntament. Normalment requereix Batxiller.
Proves físiques, teòriques i psicotècniques.
🎯 ESO → Batx → oposició municipal
💼 Seguretat ciutadana i proximitat amb la població.
✖

🚒 Bombers

Requisits: Batxiller o FP. Carnet de conduir habitual.
Proves físiques exigents, teòriques i pràctiques.
🎯 ESO → FP/Batx → oposició autonòmica o municipal
💼 Emergències, rescats, protecció civil.
✖

🪖 Forces Armades

ESO → tropa · Batx → suboficials · EBAU → oficials.
Proves físiques, psicotècniques i mèdiques.
🎯 ESO → accés directe o promoció interna
💼 Carrera militar, logística, seguretat.
Web oficial
✖

🌍 Idiomes (EOI)

Accés a partir de 16 anys amb prova de nivell.
Nivells oficials: A2 · B1 · B2 · C1 · C2
Certificats oficials útils per oposicions, universitat i treball.
🎯 ESO/Batx → EOI → certificat → estudis o treball
💼 Treball internacional, oposicions, estudis superiors.
EOI GVA

🎵 Música i Dansa

Estudis en conservatoris amb prova específica.
Inclou grau professional i superior.
🎯 ESO → prova → conservatori → estudis superiors
💼 Músic, docent, arts escèniques.
✖

🖌️ Arts Plàstiques i Disseny

Cicles de grau mitjà i superior en escoles d’art.
Disseny, fotografia, il·lustració, audiovisuals.
🎯 ESO → prova → escola d’art → grau superior
💼 Dissenyador, artista, creatiu digital.
✖

⚽ Ensenyaments Esportius

Formació de tècnic esportiu amb prova física.
Especialitats: futbol, bàsquet, muntanya, etc.
🎯 ESO → prova → tècnic esportiu → especialització
💼 Entrenador, monitor esportiu, preparador físic.
